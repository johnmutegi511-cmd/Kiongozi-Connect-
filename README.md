# Kiongozi Connect

**Primary Tagline:** Jua Kiongozi Wako Amefanya Nini Leo

**Secondary Tagline:** Connecting Leaders and Citizens

## Brand Promise

Transparency. Progress. Connection.

## Vision

A national citizen-leader engagement platform starting in Meru and expanding across Kenya.

## Mission

A digital platform connecting citizens and leaders where citizens track what leaders have done today, this week and this month while leaders showcase projects, meetings and achievements.

## Features

### User Roles

- Citizens
- Leaders
- Administrators

### Main Pages

1. Home - What Your Leader Did Today
2. Leaders - Browse and discover leaders
3. Leader Profile - View leader information
4. Projects - Track leader projects
5. Project Updates - Daily, weekly, monthly updates
6. Meetings - Meeting announcements and RSVPs
7. Discussions - Community discussions
8. Community Groups - Ward, youth, farmer, business groups
9. Live Sessions - Real-time leader discussions
10. Notifications - Activity notifications
11. Profile - User profile management
12. Subscription Plans - Premium features
13. Analytics Dashboard - User analytics
14. Admin Dashboard - Admin management

### Key Features

- **Social Features:** Likes, comments, shares, bookmarks, follows
- **Projects Module:** Create, track, and update projects
- **Meetings:** Announcements, RSVP system, venue tracking
- **Community Groups:** Ward, youth, farmer, business groups with moderation
- **Live Discussions:** Real-time sessions with Q&A
- **Notifications:** Real-time activity alerts
- **Analytics:** Profile views, engagement metrics, reach scoring
- **Subscriptions:** Basic, Standard, Premium plans

## Setup Instructions

### 1. Environment Variables

Create `.env.local` with Supabase credentials

### 2. Install Dependencies

```bash
npm install
```

### 3. Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Build

```bash
npm run build
npm start
```
